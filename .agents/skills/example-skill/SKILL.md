# Example Skill (Приклад навички)

Цей файл є головним файлом навички. Він завантажується в пам'ять агента лише тоді, коли опис (description) збігається з поточним завданням.

## Структура цієї навички:
- `SKILL.md` — цей файл з інструкціями.
- `references/` — папка для статичної документації (наприклад, чеклісти, API документація).
- `scripts/` — папка для допоміжних виконуваних скриптів (bash, python, nodejs).
- `assets/` — папка для шаблонів коду та еталонних зразків.

## Інструкції для агента:
1. Завжди спочатку аналізуйте запит користувача.
2. Використовуйте скрипти з папки `scripts/` для автоматизації (якщо такі є).
3. Звіряйтеся з документацією у `references/` перед виконанням складних дій.
4. Використовуйте шаблони з `assets/` як основу для нового коду.
